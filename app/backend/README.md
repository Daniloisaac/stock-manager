# iniciando projeto

1- Clone o repositorio
* `git clone git@github.com:Daniloisaac/stock-manager.git`


   Entre na pasta do repositório que você acabou de clonar: <br>
  `cd stock-manager.git`

2- Instale as dependências
* `npm install`

3- Rode o docker
* `docker-compose up -d`
